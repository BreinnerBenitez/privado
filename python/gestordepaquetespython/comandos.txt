====================django==============
django-admin startproject (nombreproyecto) =  crear proyecto de django
python manage.py runserver =inicar el servidor
python manage.py startapp nombredelproyecto= creacion de app del proyecto
python manage.py check gestionpedidos= para chequiar que esta bien los modulos de la base de datos sin error
python manage.py makemigrations = generar migraciones
python manage.py sqlmigrate gestionpeidos 0001 = ejecutar sentencias sql para la base de datos despues de las migraciones
python manage.py migrate = migracion a la base de datos
